Instaladores del agente
🪟
Windows — Instalador automático (recomendado)
Instala el agente como servicio Windows. Descarga e instala Python si no existe, registra el equipo en Supabase y lo deja arrancando automáticamente con el sistema. Reporta estado cada 60 segundos.
1
Descarga install_windows_supabase.ps1 y nubeprint_agent_supabase.py en el mismo directorio en el equipo cliente.
2
Genera una licencia desde el panel (menú Licencias) y cópiala.
3
Abre PowerShell como Administrador y ejecuta:
powershell -ExecutionPolicy Bypass -File "install_windows_supabase.ps1" -LicenceKey "XXXX-XXXX-XXXX-XXXX"
⚙️
Windows — Instalación manual
Si prefieres controlar cada paso manualmente:
pip install psutil requests # Primer arranque (registra el equipo): python nubeprint_agent_supabase.py --registro --key "XXXX-XXXX-XXXX-XXXX" # Arranque normal (loop 60 segundos): python nubeprint_agent_supabase.py
🐧
Linux — Instalación con systemd
Instala el agente en Linux como servicio systemd.
pip3 install psutil requests # Primer arranque: python3 nubeprint_agent_supabase.py --registro --key "XXXX-XXXX-XXXX-XXXX" # Servicio con systemd (Linux): # Copia el archivo a /etc/systemd/system/nubeprint.service y habilítalo.

macOS — Método probado con entorno virtual
Usa un entorno virtual propio para evitar el bloqueo PEP 668 de Homebrew y registra el agente como LaunchAgent de usuario.
launchctl unload "$HOME/Library/LaunchAgents/com.nubeprint.agent.plist" 2>/dev/null || true INSTALL_DIR="$HOME/Library/Application Support/NubePrintAgent" PLIST="$HOME/Library/LaunchAgents/com.nubeprint.agent.plist" PYTHON_BIN="$(command -v python3)" VENV_DIR="$INSTALL_DIR/.venv" VENV_PY="$VENV_DIR/bin/python" mkdir -p "$INSTALL_DIR" cp nubeprint_agent_supabase.py "$INSTALL_DIR/nubeprint_agent_supabase.py" "$PYTHON_BIN" -m venv "$VENV_DIR" "$VENV_PY" -m pip install --upgrade pip psutil requests "$VENV_PY" "$INSTALL_DIR/nubeprint_agent_supabase.py" --registro --key "XXXX-XXXX-XXXX-XXXX" --once cat > "$PLIST" <<PLIST <?xml version="1.0" encoding="UTF-8"?> <!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd"> <plist version="1.0"> <dict> <key>Label</key><string>com.nubeprint.agent</string> <key>ProgramArguments</key> <array> <string>$VENV_PY</string> <string>$INSTALL_DIR/nubeprint_agent_supabase.py</string> </array> <key>RunAtLoad</key><true/> <key>KeepAlive</key><true/> <key>StandardOutPath</key><string>$INSTALL_DIR/agent.log</string> <key>StandardErrorPath</key><string>$INSTALL_DIR/agent.err</string> </dict> </plist> PLIST launchctl load "$PLIST"
